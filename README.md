# dotfiles

## Installation

```bash
$ sudo apt install -y git
$ cd ~
$ git clone https://github.com/gifsyn/dotfiles.git
$ cd dotfiles
$ bash setup.sh ubuntu-22.04.conf
```

## Supported OS

- Ubuntu 22.04 LTS
- Ubuntu 22.04 LTS on WSL2

## Supported Applications

### APT Packages

- vim
- curl
- wget

### Input Method
